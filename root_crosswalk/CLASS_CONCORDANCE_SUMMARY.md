## Notes
- Class extracted from MW `cp` digits (`1P,1Ā`→{1}); Whitney roman→arabic.
- MW `cp="0P"` (class 0 = "no gaṇa assigned", pada-only) is dropped as a
  sentinel — so a 0-only root is `mw_empty`, not a spurious conflict.
- Hub classes are unioned over homonyms sharing a bare root, so the test is
  lenient (agreement if MW class ∈ any homonym's classes); true conflicts are
  therefore conservative — real disagreements, not homonym artefacts.
- Conflicts are candidates for the Whitney class-verdict review and a P4 signal;
  the Dhātupāṭha (Westergaard) class can adjudicate where present.
