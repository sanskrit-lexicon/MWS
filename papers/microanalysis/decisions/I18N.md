## Decision 3 — Bilingual labels: English + Russian

Visualisations carry labels in English and Russian — *not Sanskrit*. Sanskrit is the meta-level (the language being described, not the labelling language). Russian is the second target audience: some findings additional to the IJL paper will be published in Russian-language venues (*Индоевропейское языкознание*, the Roerich/Zograf Readings, *Petersburg Indological Studies*).

Implementation:
- Labels live in `figures/locales/en.json` and `figures/locales/ru.json`, keyed by i18n identifier.
- Static renderer (Python) accepts a locale flag.
- [csl-atlas](https://github.com/sanskrit-lexicon/csl-atlas) microsite has a locale switcher.
- Sanskrit terms are preserved in their conventional rendering (IAST in italic, SLP1 in code) regardless of locale.

## Decision 6 — Sanskrit-in-Russian: IAST in italics

When Sanskrit terms appear inside Russian text or labels, they are rendered as **IAST in italics** (*aṃśa*, *kaṣāya*, *ṛṣi*) — exactly as in the English text. **No Cyrillic transliteration**, no Devanagari.

This is the standard Russian indological convention used in:
- *Индоевропейское языкознание* (Indo-European Linguistics journal)
- *Petersburg Indological Studies*
- The Roerich/Zograf Readings
- Most major Russian-language Sanskrit grammars and primary sources

Cleanest implementation: no transcoding burden; the locale switcher does NOT transcode Sanskrit, only the surrounding prose changes.

## Decision 8 — Mermaid i18n: one file per locale

For Mermaid diagrams (timeline, lineage forest), strategy is **one `.md` file per locale**, not two Mermaid blocks in a single file.

File layout:

```
papers/microanalysis/figures/
  timeline-en.md         single Mermaid block, English labels
  timeline-ru.md         single Mermaid block, Russian labels
  lineage-forest-en.md   ditto (TBD)
  lineage-forest-ru.md

```

Cross-linking via plain Markdown link in each file's footer: `[Русская версия →](../figures/timeline-ru.md)` / `[← English version](../figures/timeline-en.md)`. Each file is self-contained Markdown that GitHub renders natively.

Rationale (vs alternative parallel-blocks approach):
- Cleaner: each file presents the diagram in one language only.
- Linkable: each locale gets its own canonical URL.
- Maintenance: equivalent (two file updates per change) but more readable source.
- Friendly to future build-step that generates `timeline-{locale}.md` from a single data file + locale-specific labels.

## Decision 11 — Russian translations: bootstrap-and-correct

Russian labels are produced by **bootstrap-from-academic-Russian-indological-terminology**, then reviewed by [@gasyoun](https://github.com/gasyoun). The terminology base draws on Zaliznyak's grammatical terminology + conventions from *Индоевропейское языкознание*, *Petersburg Indological Studies*, Zograf/Roerich Readings.

Sample bootstrap mapping (in `figures/locales/ru.json`):

| English | Russian (bootstrap) | Confidence |
|---|---|---|
| Verbal root | глагольный корень | high |
| Masculine noun | существительное мужского рода | high |
| Adjective (mfn.) | прилагательное (mfn.) | high |
| Indeclinable | неизменяемое слово | high |
| Compound | сложное слово / композит | medium |
| Derived form | производное / дериват | high |
| Continuation sense | продолжение значения | medium |
| Lexicographer-only | только в кошах (?) | low |
| IE-etymological | индоевропейское этимологическое | high |
| Botanical | ботанический | high |
| Biographical | биографический / именной | medium |
| Vedic accented | ведийское с ударением | medium |
| Block | блок | high |
| Slot | позиция / слот | medium |
| Profile | профиль | high |
| Hedge | ограничитель | medium |
| Infrastructure | инфраструктура | high |

Low-confidence terms flagged with `(?)` in [`ru.json`](../figures/locales/ru.json) for [@gasyoun](https://github.com/gasyoun)'s correction.

## Decision 25 — Default locale: English canonical

[csl-atlas](https://github.com/sanskrit-lexicon/csl-atlas) URL structure:
- `/` (root) = English content
- `/ru/` prefix for Russian variants
- Language switcher in nav: `EN | RU` toggle, persists in localStorage
- `hreflang` tags in `<head>` for SEO

Sample paths:

```
/                        landing (EN)
/tools/heatmap           18×14 matrix explorer (EN)
/tools/lineage-sankey    PWG→MW Sankey (EN)
/papers/wiegand          Wiegand paper page (EN)
/ru/                     landing (RU)
/ru/tools/heatmap        18×14 matrix explorer (RU)

```
