MONIER.ALL  From Thomas Malten. Nov 30, 2004
 E:\SANSKRIT\MONIER\MONIER.ALL, Last update 30.11.04
This is the starting point for work by Scharf, Hyman, Funderburk on
Malten's digitiztion of Monier-Williams Sanskrit-English dictionary of 1899.

MONIER.ALL is in cp1252 encoding. A program converts it to utf-8 encoding

python cp1252-to-utf8.py MONIER.ALL mw_orig_utf8.txt
